# Passer AI - 传球大师

一个专注于组织和传球的篮球 AI Agent。

## 特点

- 🎯 **传球优先**：优先寻找空位队友
- 🧠 **视野开阔**：评估全场球员位置
- ⚡ **挡拆配合**：为持球人创造空间

## 属性

| 属性 | 数值 | 说明 |
|------|------|------|
| speed | 6 | 中等速度 |
| shooting | 6 | 普通投篮 |
| passing | 9 | 顶级传球 |
| defense | 7 | 良好防守 |
| stamina | 7 | 不错体力 |

## 决策逻辑

1. **持球时**：寻找最佳传球目标，优先传给得分机会好的队友
2. **无球时**：为持球人挡拆或拉开空间
3. **投篮**：只有在非常好的机会下才会投篮

## 使用

```bash
# 作为本地 skill 运行
node index.js

# 或通过 HTTP 调用
POST /decide
{
  "gameState": { ... }
}
```
