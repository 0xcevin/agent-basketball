# Smart AI - 聪明型球员

一个根据比赛情况灵活决策的篮球 AI Agent。

## 特点

- 🧠 **智能决策**：根据防守压力、位置、队友状态综合判断
- 🎯 **攻防兼备**：持球时进攻，无球时防守或跑位
- 🤝 **团队配合**：被包夹时寻找最佳传球目标

## 属性

| 属性 | 数值 | 说明 |
|------|------|------|
| speed | 7 | 较快 |
| shooting | 7 | 良好投篮 |
| passing | 7 | 良好传球 |
| defense | 7 | 良好防守 |
| stamina | 7 | 良好体力 |

## 决策逻辑

### 持球时
1. **防守压力小 + 位置好** → 投篮
2. **被包夹** → 传给最佳队友（考虑距离、篮筐距离、防守压力）
3. **其他情况** → 向篮筐移动

### 无球时
1. **队友持球且离太远** → 靠近接应
2. **队友持球且位置合适** → 跑位寻找空位
3. **对方持球** → 防守或抢断

## 输入/输出

**输入** (GAME_STATE):
```json
{
  "me": { "id", "name", "position", "stats" },
  "teammates": [...],
  "opponents": [...],
  "ballPosition": { "x", "y" },
  "iHaveBall": boolean
}
```

**输出**:
```json
{ "type": "MOVE|PASS|SHOOT|DEFEND|STEAL", ... }
```
